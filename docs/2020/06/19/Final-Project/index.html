Final Project

Intention

I (and together with my roommates) had this experience: You bought a lot of games on Steam or Epic, but due to heavy school work, you do not have time to play those games.

So I decided to make a timer such that it can record how much time I’ve been playing each game, and alarm me if it’s time to go to work.

Components

  • Laser-cut box (wooden and acrylic)
  • Servo motor
    • together with a 3D-printed servo actuator
  • Arduino
    • LCD 1602 screen
    • Buttons x4
    • Cables & breadboard
    • buzzer (unused)

Features

  • Half-transparent shell
    • with different cassettes
  • Timing each game
    • 7 games to choose
    • Works like a toaster; when cassette rises to the top, time is up
    • With a progress bar
  • Record functionality
    • Saves data even if power off
    • Can be cleaned
  • Dual language support
    • English and Japanese
